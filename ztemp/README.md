#### Linear Program Solver
* [GLPK](https://www.gnu.org/software/glpk/): GNU Linear Programming Kit
* [Gurobi](https://docs.gurobi.com/current/):
The state-of-the-art solver for linear programming (LP), quadratic and quadratically constrained programming (QP and QCP), and mixed-integer programming (MILP, MIQP, and MIQCP)
* CPLEX: IBM High-performance mathematical programming solver for linear programming, 
mixed integer programming and quadratic programming.
